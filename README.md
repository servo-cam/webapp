**Usage:**


1) Create a virtual environment and install dependencies:

```
python3 -m venv ./venv
source ./venv/bin/activate
pip install -r requirements.txt
```

2) Start the webserver (Flask):

```python3 start.py```

3) Open web-browser and go to address:

```http://localhost:8000```
